we are now entering the world of HTML, which is short for Hyper Text Markup Language. If you have not interacted before with "code" do not fear!! This is not the bad kind of code. THink of it more like punctuation. You know how to use quotes, commas and periods to mark up your text to make it make more sense, and this is sort of an extension of that. Just as sentences are terminated by a period and started by a capital letter, we have in web layout opening and closing "tags" which create "elements" much like sentences and paragraphs.

we begin with the heading tag h1. H is for heading and the 1 denotes the BIGGEST type of heading, which is the only one we use in TRASH MAGIC web design. HTML tags are always between the less than and greater than signs, and the closing tag has a forward slash before the symbol/letter, which is case insensitive. This means that H1 and h1 both work the same. Put the cursor between the tags of the title in your new page and change it to whatever is appropriate, preferably in ALL CAPS for design consistency. This title will be automatically rainbow-ified by the Trash Magic web code when it appears in a browser.

The next HTML element we will use is "p" which is for "paragraph". Paragraphs, like the one this text lives in, are rendered in Trash Magic as blue boxes with white letters. Edit the text to whatever you want, some type of introduction about who you are and what you're all about on your page. You can add another paragraph by either scrolling down and putting in another p tag(always in between angle brackets) or you can copy/paste the existing paragraph and change that so that you don't have to type the tags again.

The third and final type of tag we will use here is the link, which is confusingly an "a" tag. "A" stands for "anchor", and while it might seem more ligical for it to be "L" for link or "H" for hyperlink, "a" is what was chosen and so it remains. This tag is more complex than the other two. It has something called an "attribute", which in this case is the address that the link is pointing to, which the browser will jump to if a user clicks on the link. The attribute here is called "href". It takes a little getting used to, we recommend that you just copy/paste the links and change what's in quotes after href to whatever address you want to link to and change the words between the a opening and closing tags to the text you want.

Use the link copy/paste procedure to add links to whatever you want, like a link tree. Put links to your social media, links to contact info, links to home pages, links to pages you like, and links to your friends. Use simple text, for example you can put in your whole exact web address for your tiktok account but then the text will look simple if it just reads "TIKTOK" in the text. All these links will also get rainbow-ified by the Trash Magic web code automatically.
