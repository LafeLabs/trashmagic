CARDBOARD

Cardboard is an essential element of Trash Magic. It is the prototypical free medium. Cardboard trash can be found anywhere. It can be turned into media with just a magic marker. It can be turned into a wide range of complex objects with a box cutter, geometry tools, markers, and glue. It can be turned into many identical things quickly using a laser cutter.

We use cardboard as an extension of the Web Browser. We write web page addresses on cardboard which point to web servers running trash magic which in turn point back to the cardboard out in a shared physical space. This makes everything recursive. We also make reference here to the expression used in alchemy and related arts "as above, so below". In the context of our system, this can refer to the correspondence between the information we put in HTML on web servers and the information we put out in the physical world.
